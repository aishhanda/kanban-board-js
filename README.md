# 🧩 Kanban Board

A simple and clean Kanban Board built using **HTML, CSS, and Vanilla JavaScript**.

## 🚀 Features
- Drag & Drop tasks between columns
- Add / Delete tasks
- Persistent data using LocalStorage
- Dynamic task counters
- Clean glassmorphism UI

## 🧠 What I Learned
- DOM manipulation (createElement, appendChild)
- Drag & Drop API (dragstart, dragover, drop)
- Event bubbling & event.target vs currentTarget
- LocalStorage for saving app state
- Structuring code into reusable functions
- Improving UI/UX with transitions and animations

## 📦 Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## 🔧 How to Run
1. Clone the repo
2. Open `index.html` in browser
